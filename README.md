# Calculator Project (Python + Pandas)

This is a simple calculator project built using **Python** and **Pandas**.  
The program takes multiple rows of numeric input from the user and performs the selected operation:

- Addition  
- Subtraction  
- Multiplication  
- Division  

The final result is displayed in a **Pandas DataFrame** format.

## Features
✔ Takes any number of inputs  
✔ Supports 4 operations  
✔ Uses Pandas for DataFrame representation  
✔ Beginner–friendly project
